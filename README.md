## 😎 개요

- Google 혹은 Github 계정을 통해 로그인하여 명함을 작성하고 저장하는 사이트입니다.
- Social Login (Google, Github) , Logout
- 개인 명함 작성, 저장 메인 화면
- 내용 수정과 동시에 저장 반영
- 사진 업로드

## 😊 기술스택

- HTML / Post CSS
- React Hooks
- React Router
- Firebase
- Cloudinary
